# La Memoria

Un microcontrolador tiene dos componentes de almacenamiento que funcionan de forma distinta. Estos son:

###  **1.** Memoria FLASH 

Este es como si fuera el disco duro de un PC, pero en microcontrolador.
Se llama no volátil porque cuando le quitas la corriente no se borra nada.
Aquí se guarda todo el código binario, las funciones, constantes...


###  **2.** Memoria RAM (SRAM)

Esta memoria sí es volátil, lo que significa que cuando le quitas la corriente, se borra todo.  
Esta memoria es como la mesa donde trabaja, donde todo lo que usa lo pone aquí para trabajar con ello.

La diferencia de esta memoria es que es más rápida que la FLASH.
